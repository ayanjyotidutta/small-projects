# 🗺️ User Journey Map: OpticAI
**Status:** Version 1.0 (Official Release — High-Fidelity Benchmarking)

---

## 1. The Entry Point (The Benchmark Initiation)
* **Touchpoint:** Landing Dashboard.
* **Visual Strategy:** **The Universal Glow CTA.** By implementing the `@glow-pulse` animation, the UI evolves from a simple "Start" to a high-intensity pulse that mimics a digital heartbeat against the #000 background.
* **Tactile UX:** User clicks "Start OpticAI" — the `active:scale-95` transition provides instant haptic-style feedback, signaling the start of a professional audit.
* **Pre-Flight Calibration:** A minimalist suggestion block (*Brightness > 80% / Disable Night Shift*) ensures the user's hardware is optimized for elite-level accuracy before the first round.

## 2. The Core Loop (The Perceptual Audit)
* **Memorise (5s):** **Total Retinal Focus.** The heartbeat-synced millisecond timer drives cognitive urgency. The addition of the "Memorise the Colour" header anchors the user’s mental state in pure observation.
* **Guess (30s):** **Precision Calibration.** User manipulates the **Glassmorphic Control Panel**. Sliders are protected by `touch-action: pan-x` to ensure zero-latency precision on mobile devices.
* **Instructional Anchor:** The "Guess the Colour" header ensures the user never loses the objective in the heat of the 30s countdown.
* **The Lock:** User clicks "Lock it In." The **Ghost-Timer Fix** ensures that manual progression immediately terminates any background countdowns for a seamless transition.

## 3. The Climax (The Visual Audit)
* **Scoring & Tiering:** The system applies the **1.8x Cubic Decay Algorithm**, assigning a Tier (e.g., "Divine Visionary") that reflects professional-grade visual memory.
* **Benchmark Context:** The final score now features the **"/50" suffix**, providing immediate scale and professional authority to the result.
* **Visual Calibration:** The summary screen displays **Split-View Comparison Cards**. This is the "Learning Moment" where the user identifies their specific color bias (Hue vs. Saturation vs. Brightness).

## 4. The Exit & Viral Loop (Social Equity)
* **Touchpoint:** The **Neomorphic "Share" CTA.**
* **Branding Consistency:** By mirroring the "Start" button’s architecture (Title Case + Circular Glyph + Radiant Glow), the UX feels "complete."
* **The Elite Floor:** Retention logic is now "Prestige-Locked." Instead of just [Score + 1], it enforces a **Minimum 25-point floor** (*"Try Beating 25 Score"*), signaling that the professional competition begins at 50% accuracy.
* **Action:** User clicks "Share Result." The system bundles the Tier, Score, and Link into a "Wordle-style" text block for high-conversion social proof.

---

### 🎨 Strategic UI Framework: "The Radiant Bookend"
We implemented a consistent button architecture across the **Start** and **End** states to drive brand authority:

1.  **Symmetry:** Identical button shapes and iconography create a sense of professional "Finish."
2.  **Tactile UI:** Both buttons use dual-source shadows and hardware-style depth, elevating the tool from a "web game" to a "Benchmark Product."
3.  **High-Intensity Feedback:** The radiant white glow on both ends of the journey creates a psychological loop—the glow represents the "Active State of Benchmarking."
